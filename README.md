# LLM Prompt Package

## 프로젝트 소개
본 프로젝트는 LLM 기반 업무 자동화 프롬프트 설계 과제로, 동일한 업무 과업을 GPT, Claude, Gemini에 적용하여 성능을 비교하고, 최적의 모델을 선정하기 위해 수행되었다.

## 업무 과업
보고서 개요 자동 생성

## 연구 주제
DED 공정에서 TiB₂ 첨가가 Al6061의 미세조직과 기계적 특성에 미치는 영향

## 작성 목적

- 동일 프롬프트 기반 LLM 성능 비교
- 시스템 프롬프트 설계
- Few-shot 및 Reasoning Prompt 적용
- 환각(Hallucination) 검증
- 10턴 이상 문맥 유지 검증
