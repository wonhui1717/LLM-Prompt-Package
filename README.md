# LLM Prompt Package

## 프로젝트 개요

본 프로젝트는 **LLM 기반 업무 자동화 프롬프트 패키지**를 설계하고, 동일한 업무를 3종 이상의 대규모 언어모델(LLM)에 적용하여 성능을 비교·평가하는 것을 목표로 한다.

선택한 업무 과업은 **연구보고서 개요 자동 생성**이며, 연구자가 입력한 연구 주제를 기반으로 학술 보고서 개요를 자동 생성하는 프롬프트를 설계하였다.

---

# 프로젝트 목표

본 프로젝트의 목표는 다음과 같다.

- 동일한 업무를 3종 이상의 LLM에서 수행하여 성능을 비교한다.
- 업무 자동화에 적합한 프롬프트를 설계한다.
- 환각(Hallucination)을 줄이기 위한 프롬프트를 설계한다.
- 반복 사용 가능한 업무 자동화 프롬프트 패키지를 구축한다.

---

# 업무 과업

## 업무명

**연구보고서 개요 자동 생성**

## 업무 설명

사용자가 연구 주제를 입력하면 AI가 학술 보고서 형식의 연구 개요를 자동 생성한다.

생성 결과에는 다음 항목을 반드시 포함한다.

- 보고서 제목
- 연구 목적
- 연구의 필요성
- 핵심 키워드
- 보고서 목차
- 각 장의 핵심 내용
- 기대 효과

실험 결과나 논문에 없는 내용은 생성하지 않으며, 정보가 부족할 경우 **"확인 필요"**를 출력하도록 설계하였다.

# 결과

본 프로젝트에서는 동일한 업무 과업을 대상으로 프롬프트를 반복 개선하고, 여러 LLM을 비교 평가하여 업무 자동화에 가장 적합한 프롬프트를 설계하였다.

최종적으로 **ChatGPT(GPT-5.5)**를 연구보고서 개요 자동 생성 업무에 가장 적합한 모델로 선정하였다.

선정 이유는 다음과 같다.

1. 출력 형식 준수율이 가장 높았다.
2. 정보 부족 시 "확인 필요"를 제시하여 환각을 줄였다.
3. 동일 프롬프트에 대한 결과 일관성이 우수하였다.
4. 반복 업무 자동화에 적합한 재사용성을 보였다.
