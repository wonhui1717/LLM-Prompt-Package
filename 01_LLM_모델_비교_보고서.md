# 01. LLM 모델 비교·선정 보고서

# 1. 비교 목적

## 업무 과업

연구보고서 개요 자동 생성

## 비교 목적

동일한 연구 주제와 동일한 프롬프트를 ChatGPT(GPT-5.5), Claude Sonnet, Gemini에 입력하여 결과를 비교하였다.

비교 목적은 연구보고서 개요 자동 생성 업무에서 가장 높은 정확성과 재사용성을 제공하는 LLM을 선정하기 위함이다.

---

# 2. 업무 과업 정의

|항목|내용|
|---|---|
|업무 과업|연구보고서 개요 자동 생성|
|타겟 사용자|신소재공학과 학생, 대학원생, 정부출연연구원 연구원, R&D 연구원|
|입력 데이터|연구 주제, 출력 형식, 추가 요구사항|
|출력 규격|보고서 제목, 연구 목적, 연구의 필요성, 핵심 키워드, 목차, 각 장 핵심 내용, 기대 효과|

# 4. 동일 입력 프롬프트

모든 모델에는 아래와 동일한 프롬프트를 입력하였다.

```text
당신은 30년 경력의 정부출연연구원 연구부장이다.

다음 연구 주제를 바탕으로 학술 보고서의 개요를 작성하시오.

[연구 주제]

DED 공정에서 TiB₂ 첨가가 Al6061의 미세조직과 기계적 특성에 미치는 영향

다음 요구사항을 반드시 준수하시오.

1. 보고서 제목
2. 연구 목적
3. 연구의 필요성
4. 핵심 키워드
5. 보고서 목차
6. 각 장의 핵심 내용
7. 기대 효과
8. 실제 논문이나 실험 결과가 없는 내용은 추측하지 말 것
```

---

# 5. 평가 기준

평가는 아래 6개 항목을 기준으로 수행하였다.

|평가 항목|평가 내용|
|---|---|
|정확성|연구 주제 이해 및 사실 기반 작성|
|논리성|보고서 구조와 내용의 연결성|
|형식 준수|요구한 출력 형식 준수 여부|
|한국어 자연스러움|문장의 가독성과 표현|
|환각 억제|근거 없는 정보 생성 여부|
|업무 자동화 적합성|반복 사용 가능성과 재사용성|

---

# 6. 모델별 결과 요약

## 6.1 ChatGPT (GPT-5.5)

### 결과 요약

- 요구한 형식을 모두 충족하였다.
- 연구 목적과 목차의 논리적 연결성이 우수하였다.
- "확인 필요" 항목을 제시하여 정보 부족 시 추측을 하지 않았다.
- 반복 실행 시 출력 형식이 가장 안정적이었다.

### 장점

- 형식 준수 우수
- 환각 억제 우수
- 업무 자동화 적합성 우수

### 단점

- 일부 전문 용어 설명은 Claude보다 간결하였다.

---

## 6.2 Claude Sonnet

### 결과 요약

- 재료공학 전문 용어를 가장 풍부하게 사용하였다.
- 대학원 논문 수준의 문체를 보여주었다.
- 연구 배경과 이론적 배경 설명이 상세하였다.

### 장점

- 전문성 우수
- 논리성 우수

### 단점

- 일부 내용을 일반적인 사실처럼 단정적으로 기술하였다.
- 정보 부족 시 "확인 필요" 안내가 부족하였다.

---

## 6.3 Gemini

### 결과 요약

- 다양한 재료공학 용어를 사용하였다.
- Hall-Petch, Orowan 강화 등 심화 내용을 적극 포함하였다.

### 장점

- 전문 용어 활용 우수

### 단점

- 요구하지 않은 내용을 추가하였다.
- 출력 형식이 일부 일관되지 않았다.
- 정보 부족 시 추측성 설명이 포함될 가능성이 있었다.

|자동화 적합성|반복 사용 시 결과 일관|출력 길이 변동 존재|출력 형식 변동 존재|

# 9. 최종 선정

## 선정 모델

**ChatGPT (GPT-5.5)**

### 선정 근거

첫째, 동일한 프롬프트를 반복 입력하였을 때 출력 형식의 일관성이 가장 우수하였다.

둘째, 정보가 부족한 경우 "확인 필요"를 제시하여 실제 실험 결과나 수치를 임의로 생성하지 않아 환각 발생 가능성이 가장 낮았다.

셋째, 보고서 개요 자동 생성 업무는 정확성과 재사용성이 중요하므로 반복 업무 자동화에 가장 적합한 모델로 판단하였다.

---

# 10. 결론

세 모델 모두 연구보고서 개요를 생성할 수 있었지만 업무 자동화 관점에서는 차이가 있었다.

Claude는 전문성이 가장 높았으나 일부 단정적인 표현이 존재하였다.

Gemini는 다양한 전문 용어를 활용하였지만 형식 준수와 환각 억제 측면에서 개선이 필요하였다.

ChatGPT는 형식 준수, 정확성, 환각 억제 및 재사용성 측면에서 가장 우수한 결과를 보여 연구보고서 개요 자동 생성 업무에 가장 적합한 모델로 선정하였다.
